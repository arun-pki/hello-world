Autosave and Autorecovery

You can configure ChemDraw to periodically save an open document. In addition, a default Enable Autorecovery checkbox saves a temp file if the program crashes.
Note: You will be allowed to autosave/autorecover documents having unset required values for document properties. However, you will not be able to save the auto recovered document if there is an unset required document property. You can set the value for the required document property in the Document Properties dialog and Save.
You can specify where the autosaved or autorecovery copies are stored.
1. Navigate to File>Preferences. The Preferences dialog appears.
2. Click the Open/Save tab.
Note: If using a Macintosh, navigate to ChemDrawProfessional/ChemDraw Prime > Preferences...
3. Select the Save Files Every check box and enter a time frame (in minutes), for example 5 minutes.
The time starts after you make your first change to the document.
