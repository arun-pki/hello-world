Property Predictions in ChemDraw/Excel

The Properties feature in ChemDraw/Excel lets you view the values of different properties for a given structure. Each property is calculated using the “best available” method. A list of properties available in ChemDraw/Excel is listed in the table below. For more information on these properties, see ChemDraw/Excel online help, or visit the PerkinElmer Informatics website .
The properties available include:
