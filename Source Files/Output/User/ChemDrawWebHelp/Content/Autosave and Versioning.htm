Auto Save and Versioning

By default, native files are saved automatically as you edit them. You will not be prompted to save changes. You can change this behavior using the General system preferences.

Non-native file formats, (not CDX or CDXML) are autosaved in the System Autosave folder for the application as CDX files to avoid potential data loss.

Using Save As

"Save As" is not available by default. However, you can save a copy of the file with a different name using Duplicate sub menu option.
To save a copy of a file with a different name:
1. Go to File > Duplicate.
2. Enter the file name.
3. Click Save.
Note: When the 'File' menu is open, holding the 'OPTION' key will display 'Save As' option instead of 'Duplicate' option.
