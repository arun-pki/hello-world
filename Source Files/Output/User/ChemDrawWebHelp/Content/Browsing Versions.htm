Browsing Versions

The 'Revert to ' option lets you browse through all saved versions of a document and revert the changes to the desired version.
To do so:
1. Go to File > Revert to > Browse All Versions. All versions of the saved file appears.
2. Select the version you want to restore and click the Restore button.
3. To close the dialog without restoring, click Done.
