Editing Scripts

Using IDLE or another development environment, you can either edit the scripts provided with ChemOffice or create your own. Regardless of how you develop a script, it must include these commands:
from sys import *
from os import *
from os.path import *
from ChemScript17 import *
The first command imports the python system. The second and third commands import the operating system modules. The last command imports all the ChemScript functions. After you include these command lines, how you develop the rest of your script is entirely up to you.
