Accessing the Custom Monomer Editor

You can choose to create your own monomers with unique symbol, full name and structure with proper capping group. You can use this option to maintain a set of custom monomers you use frequently.
To access the Custom Monomer Editor:
1. Select Custom as the Data Source type in the HELM Monomers toolbar.
2. Click on the Open Monomer Editor button. The Monomer Editor dialog appears.

3. Select the polymer type.
4. Specify a unique Symbol and Full name for the monomer.
5. Select a Natural analog and Monomer type (backbone /branch).
6. Draw or paste the structure and select a label for collapsed state.
7. Specify the label to be used when the capping group is in collapsed state.
8. Click Save to Custom set.
Note: The combination of structure, capping groups and symbol must be unique within any polymer type. For example, it is not permitted to have two peptide monomers with the same template structure and capping groups.
9. Click Add New to continue adding monomers to custom set.
The custom monomers added are displayed in the HELM Monomers Toolbar under appropriate polymer types.
