Accessing the Imported Monomer Editor

You can also import a HELM string from a cdxml file containing it. You can then save it to the custom set.
Note: Monomers will only be added to the imported set if they do not appear in the Standard or Custom set.
To do so:
1. Double-click on the cdxml file. The following message appears if there are monomer present that are neither in the standard or custom set.
2. Click OK to close the message box or click Open Imported Monomer Editor... to display the Monomer Importer dialog.

3. Click on Save to custom set button to add the monomer to custom set.
