Exporting tables

ChemDraw/Excel lets you export tables to MDL SDFiles format. The first column of the area you select to export must have the word “Structure” in its top cell. Structures outside the first column are discarded.
To export data:
Select the cells to export, including the cell that contains the word “Structure”.
You can export selected rows and columns from a structure table. Ensure that the cell selection is continuous. See examples below.
Go to ChemOffice17>Import/Export>Export Table.
In the Save Table To dialog box, type the file name and click Save. The status bar displays the progress.
