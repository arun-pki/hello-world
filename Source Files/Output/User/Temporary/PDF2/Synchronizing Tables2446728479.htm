Synchronizing Tables

You can synchronize a table in your worksheet with a ChemFinder database, for example, a .CFX file. On synchronizing a ChemDraw/table with a ChemFinder table, fields (and their data) in the ChemFinder table that are not in the MS Excel table are added to the MS Excel table.
Changes you make in MS Excel do not affect the table in ChemFinder.
To synchronize a table:
Select the table.
Go to ChemOffice17>Sync Table with ChemFinder DB.
Select the ChemFinder table and click Open.
Select the fields to be used for matching a worksheet row with the database record and click OK.
Select the fields to be updated. To select multiple fields, press <Ctrl> + click each field. Selecting fields marked ‘New’ will add a column to your worksheet.
Click OK.
