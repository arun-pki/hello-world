Stereochemistry

ChemDraw calculates the absolute stereochemistry according to the Cahn-Ingold-Prelog (CIP) priority rules. For more information, see "Cahn-Ingold-Prelog" on page 1.
Only tetrahedral and double-bond stereochemistry are supported, and only non-racemic stereochemistry is interpreted. Stereochemical indicators for aromatic bonds are not displayed.
ChemDraw uses this notation:
(R), (S)
Standard tetrahedral stereochemistry
(r), (s)
Tetrahedral stereochemistry determined by other stereochemical centers. For example: cis-decalin and myo-inositol
(E), (Z)
Standard double-bond stereochemistry
