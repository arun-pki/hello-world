Working with Color

You can create full color presentations of your chemical drawings to appear on your monitor, print on a color printer, or create 35mm slides using a film printer.
Most computers can display up to 16 million colors, but the number of colors may be limited by your monitor and display card.

What You Can Color

In ChemDraw , you can change the color of the background, foreground, or individual objects and elements:
Background color
The color of the document window. By default, the background is white.
Foreground color
The color of objects that you draw. By default, the foreground is black.
Individual objects
You can assign colors to objects to help then stand out from other objects.
Individual elements
You can assign colors to specific atoms or elements in a structure.
To change the background and foreground colors:
Navigate to File>Document Settings and select the Colors tab.
Select the background and foreground colors.
Click OK.

Color by Element

You can now color atom labels using colors associated with their atomic symbols.You can assign colors to elements in the periodic table. Whenever you use a particular element in your drawing, the element appears in same color you assigned to it in the periodic table.
To assign different colors to elements in a periodic table:
Navigate to View>Show Periodic Table Window. The periodic table appears.

Double-click an element. The color palette appears.
Select a color and click OK.
The modified colors are saved when you exit ChemDraw.
To apply a color to a structure:
Select the structure.
Navigate to Color>By Element.

 

The atom colors are not dynamically set, hence changing the label does not reset the color.

The Color Palette

Use the color palette to specify the color of objects and text in the document in the active window. A palette is stored in every document and style sheet.You can use style sheets to create a series of documents with the same color scheme.

Customizing Colors

Changes you make to the color palette affect the current document only. You can save up to 20 colors. For more information, see "Document and Object Settings" on page 1.
If you select an object or group that contains multiple colors, a check mark appears next to each of the colors in the Color menu. The check mark alerts you that a change would affect more than one color.
To change the color palette in the current document:
Navigate to File>Document Settings. The Document Settings dialog appears.
Click the Colors tab.
Click the color to change (either background or foreground) and select Other. The Color dialog opens.
Click the new color in the Basic Colors or the Custom Colors section.
Click OK. The color changes to the new color and appears on the Color menu.
To add a customized color:

 

Setting custom colors. A) Color refiner box; B) Luminosity box.

Click a color in the Color Refiner box to set the hue and saturation.
The pointer turns into a cross hair when you click. You can drag to a different region to change the hue and saturation.
Click in the Luminosity box to set the color brightness.
If necessary, change the hue, saturation, luminosity, and RGB components by typing the values in the text boxes.
Click Add to Custom Colors.
Click OK.

Templates and Color

The background and foreground colors in a template from the template pop‑up palette are not used when the template is drawn in a document window. However, other colors in the template are added to the color palette of the document if they are not already present. For more information, see "Autoscaling" on page 1.

Saving Color Palette Settings

You can save the color palette in a style sheet. The color palette is saved with other document settings such as page setup settings, text settings, and drawing settings.

Printing Background Color

To change whether the background color is printed:
Navigate to File>Preferences.
Select or de-select Print Background Color on the General tab.
Click OK.
