Uploading Documents to ChemDraw Cloud

To upload documents from ChemDraw to the ChemDraw Cloud:
1. Click File > ChemDraw Cloud and select Uploadâ€¦
2. Log in with valid account.
Note: if you are already logged in, the Login page will not appear.
3. Enter the document name and click Upload to save the document to the ChemDraw Cloud.

If the specified document name already exists, the following dialog appears:

You can chose to do any one of the following:

Click Cancel to cancel upload.
Click Browse to browse for existing documents and select one to overwrite.

Overwrite a Document

There are two ways you can overwrite a document:
Select a document from the document list view.
Enter keywords in the Search box to display a suggested list.
To overwrite a document from the document list view:
1. Select the down arrow link in the top right of the page to view a document list:

2. Select the document in the list that you want to overwrite.
Note: The Upload button changes to Overwrite.

3. Click Overwrite. Click Overwrite again to confirm it.

 
To overwrite a document by entering keywords:
1. Enter some appropriate keywords in the search box to display a suggested list.

2. Click one of the items from the suggested list and click Overwrite. A confirmation message similar to the one in Step 3 above appears.
3. Click Overwrite.
