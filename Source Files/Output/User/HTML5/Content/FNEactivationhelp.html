Help for PerkinElmer Product Activation

 

Activate your software (Online)

Type your name, email address and activation code into the designated fields and click on the "Activate" button.  If the activation succeeds, you will receive a confirmation message. 

 

Problems with Activation 

If activation was unsuccessful, there are several possible reasons:

 

*  A firewall may be blocking access to the activation server.  If possible, temporarily disable your firewall and then try the activation again.

*  Your computer may need to install an SSL certificate.  You will get a special message with a solution to this problem directing you to our website, https://perkinelmer.flexnetoperations.com.  Simply visiting the site will provide you with the certificate.

*  You might not be connected to the internet.  If so, try again after reconnecting to the internet.

*  The activation server might be down or temporarily busy.  In this case, try again later.

 

Activation issues experienced in ChemOffice/ChemDraw 17.0/17.1

Some windows users experienced activation difficulties with v17.0/17.1 due to having a non-unique HostID.

This could manifest in a few ways such as unexpectedly:

*being presented with the activation dialog after activating.

*being able to use the software without activating and seeing an unexpected Activation Code in the activation dialog.

These issues may have been resolved by locally altering the HostID setting to avoid these issues, which is recorded in the file:

C:\ProgramData\PerkinElmerInformatics\FNE\Settings.ini

If so, then that existing resolution is all that is required.

 

The windows default HostID (used by the Activation process) in ChemOffice/ChemDraw v18.0 is now generated by a method much more likely than that used in 17.0/17.1 to ensure it is unique.

If v18.0 is installed on a windows machine on which v17.0/v17.1 was never installed, this new HostID is used by default.

Otherwise, when v18.0 is installed, the following should be performed to switch to and use the new HostID.

1) Deactivate any existing license for v17.0/17.1 by launching an application for e.g. ChemDraw and selecting the menu option: Help>Activate ChemDraw ... and then clicking on "Deactivate" button.

2) Open the following file in a text editor e.g. NotePad:

C:\ProgramData\PerkinElmerInformatics\FNE\Settings.ini

Delete the 1st line which is of the form:

<

HostID=<Current HostID>

>

3) Save the file and exit the editor.

4) v18.0 can now be installed. If installed via Install.exe, or via .msi (when the first ChemOffice product e.g. ChemDraw is launched), the new HostID will be generated and the Activation dialog appears. The software can now be activated.

If the online activation failure still persists, you can choose one of the alternative activation methods described below by clicking on the "More Options >>" button (for Windows) or expanding the down arrow next to "More Options" (for Mac).

 

Activate by uploading a file

*  Select the "Activate or deactivate by uploading a file" option and click on the "Activate" button.  You will be prompted to save a local .BIN file to some location you choose (e.g., your Desktop).  Then close the Activation Dialog.  Continue installing the software to completion, if you haven't done so already.

*  Navigate to the following web site: @@fnologinurl and log in. If you have never logged in before, create an account using your organization’s email address. Once logged in, select "Manage Hosts">"Search Devices" from the menu bar at the top of this web page. Copy and paste the "Host ID" for the computer you wish to activate on (available from the Software Activation Dialog - for this computer, it is: @@fnehostid) in the "Device ID" field and then click "Search".  Next click on the Device ID link to open the "View Device" page.  If this is not found it simply means that no activation has yet been attempted for this computer - a record can be created by selecting "Manage Hosts">"Add Device" from the menu bar at the top of the web page, copying/pasting your "Host ID" into the "Device ID" field and clicking "Add device".

* Once the HostID for the computer you are trying to activate on has been successfully added, click on the Device ID to open the View Device page, and select the "Choose File" (or "Browse") option.  Navigate to your .BIN file and then select "Upload capability request" to upload this .BIN file.  This results in a "Response.BIN" file downloaded from the site to your computer.  This file should be placed in the @@fnetsfolder folder on your hard drive.  Then relaunch the software to activate the product.

Activate later

If you choose not to activate now (or are unable to because of a firewall or lack of an Internet connection etc.), you have a number of free launches of the product available before activation is required.  Select the "Activate later" option and click on the "Activate" button to continue.

 

Deactivation of the software

Each activation obtains a license to use the software for your computer.  Licenses can be moved to other computers.  This can be achieved by deactivating the software on a currently licensed computer and activating it on another.  This operation is especially important if only a few license seats have been purchased.  You can choose to deactivate the product by selecting either “Activate or Deactivate online” option or “Activate or deactivate by uploading a file” option. Follow the steps mentioned above for activating the product and click "Deactivate" instead of "Activate".
