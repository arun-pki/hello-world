Accessing the Monomer Viewer

Displays monomer details from the monomer libraries with Read-Only access for eg. PerkinElmer.
To access the Monomer Viewer for PerkinElmer monomer library:
Select PerkinElmer as the Data Source type in the HELM Monomers toolbar,
Click on the Open Monomer Editor button. The Monomer Viewer dialog appears.

The monomers are displayed in view only mode for the PerkinElmer monomers and cannot be edited.
You can choose to copy the monomer to a different library by selecting the desired monomer library and selecting the "Copy' option.

Filtering Monomers

You can filter monomers using any of the following three fields:
Symbol/Name
Natural Analog
Substructure Query
You can also filter by symbol and monomer name by typing in the spotlight search field.
To filter monomers:
Select the polymer type:
Peptide- amino acids
RNA- both DNA and RNA
Chemistry- Chem
Blob
Click on the button shown in the image below to bring up the Monomer Filters dialog:

The Monomer Filters dialog appears.
Enter a Symbol/Name and/or select a Natural analog and/or click within the Substructure query field to draw a substructure to query.
The search is performed based on the values specified in the Symbol/Name, Natural analog and Substructure query fields. If more than one fields are specified, the records that satisfy all the criterion specified are retrieved.
Click Search.
The filtered monomers are listed in the Editor Window.

Click on the Reset Filter button to specify a new filter criteria.
