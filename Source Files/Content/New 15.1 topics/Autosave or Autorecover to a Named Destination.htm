Autosave or Autorecover to a Named Destination

You can select an Autosave or Autorecovery path for your files.
In Windows:
Navigate to File>Preferences. The Preferences dialog appears.
Click the Directories tab.
Select the Use Documents Location check box to save files to the default location, or click the Browse button next to the Location for AutoSave/Autorecovery Path field and specify a location for your saved and recovered files.
