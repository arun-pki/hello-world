Working with ChemDraw Files in Elements

Registered Elements users can open CDXML files in ChemDraw which were created by the Elements application. Specifically, the user can browse to their Elements CDXML file from within ChemDraw, modify the file (or not), rename the file, and then save it locally or upload it to back to the Elements cloud environment. Note that the instructions that follow apply to both Windows and Macintosh operating systems.
To use Elements, Windows users must be working with ChemOffice Professional. Macintosh users must be working with ChemDraw Professional.
In addition, registered Elements users can create new CDXML files within ChemDraw, name them, and then upload them to the Elements.
If you create a ChemDraw file with the desktop application, some of the features of the drawing may not display in Elements because Elements currently only works with ChemDraw Direct, which does not have all the capabilities of desktop ChemDraw.

Browsing Elements

To locate the CDXML file you want to work with:
Click the Browse Elements icon on the General toolbar, or
Click File>Cloud>Browse Elements...
The first time you browse Elements, you must sign in to the Elements application or create an Elements account for yourself.

To browse Elements:
Enter your Elements user name and password, and then click Sign In.
Scroll through the document list below, select the title you want to view, and then click Open. You can also double click on the title. The CDXML file opens with the ChemDraw application.
You can also search for a CDXML file. Enter a minimum of the first three letters of the title of the CDXML file you want to open in the Search field, and then click the magnifying glass or press Enter.
Edit the files as you want. If you save the file it will update in the Elements environment. If you Save As..., you can save locally or create another copy of the file in the Elements environment.

Uploading ChemDraw files to Elements

Follow these instructions if you want to upload any newly created CDXML files to the Elements application via the cloud. These can be files you created from existing Elements CDXML files or newly created ChemDraw files.
Click File>Cloud>Upload to Elements ...
Enter the name of the CDXML file you want to save to the cloud, and then click Upload.
Your CDXML file saves if it does not have a duplicate name. If a duplicate name exists, continue to Step 3.
Choose from the following options:
Click Cancel to cancel the upload operation
Click Browse to check for a duplicate name
Click Continue to save the CDXML file (with a new timestamp)

 
Unlike your local machine, Elements will allow multiple files with identical names.

Overwriting an existing CDXML file name

Click the down arrow on the top right of the screen to view documents in a list format.
Select the document in the list you want to overwrite. Notice an Overwrite button displays at the bottom right of the screen.
Click the Overwrite button.

Click the Overwrite button in the Confirm to Overwrite? section of the screen. Your selected CDXML file is now overwritten.
