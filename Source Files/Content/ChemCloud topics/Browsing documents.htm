Browsing Documents in ChemDraw Cloud

To browse for documents in the ChemDraw Cloud:

Click File > ChemDraw Cloud, select Browse... and log in with valid account. The Browse ChemDraw Cloud page appears.

From this page you can:
Open documents
Sort documents
Page through documents
Search documents

Opening Documents

You can open a document in one of the following three ways:
Click one row in the list, and then click the Open button.

Double click one row in the list.
Type characters in the search box to open a dropdown list. Click one item from the list.

Sorting Documents

To sort documents, click on the desired header. The "Last Modified" header is sorted by default.

The "Name" and "Owner" column sorts by ascendant at first click
The "Last Modified" column sorts by descendant at first click.

Browsing Additional Records

The most recent records load by default. If you want to browse more records, use the scroll bars at the right.

Searching Documents

You can search documents by typing some keywords in the search box. If you do not enter a search term, all records are displayed. There are three ways to trigger a search:
Type some keywords in the search box, and then click the Search button.

 

Type some keywords in the search box, and then select "Show search results" from the dropdown list.

 

You must enter at least three characters in the search box to display the suggested search list.
Simply type keywords in the search box, and press the Enter key.
After typing characters in the search box, an ‘X’ mark displays to the right. Click ‘X’ to clear the filter.
