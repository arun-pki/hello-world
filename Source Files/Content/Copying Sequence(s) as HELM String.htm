Copying Sequence(s) as HELM String

You can now copy any biopolymer sequence(s) as a HELM String.
 
To copy biopolymer sequence(s) as HELM string:
Draw the desired biopolymer sequence(s). An example is shown below.
Select the sequence(s).
Go to Edit>Copy As>HELM.

The HELM string is transferred to the Clipboard. To display the HELM string, paste the string in the document window.
When the HELM string is in the Clipboard, you can use it in any application that can generate HELM sequences from HELM strings.
