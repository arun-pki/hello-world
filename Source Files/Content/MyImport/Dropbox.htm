Working with Dropbox Interface

ChemDraw provides an interface to Dropbox, which serves as a cloud repository for files. The Dropbox service is accessed through the ChemDraw interface. To use the Dropbox service, you must have:
A valid Dropbox account, and
ChemDraw installed
You can download a Dropbox plug-in from the Web site https://www.dropbox.com/.
You can upload your drawings from the current ChemDraw document window to your remote Dropbox folder location. You can later download these files when required.

Signing into Dropbox

When accessing Dropbox from ChemDraw for the first time, you need to authorize ChemDraw’s interaction with the Dropbox folder.
To authorize ChemDraw's interaction with Dropbox:
Navigate to File>Cloud>Dropbox>Sign In. The ‘Enter Dropbox User Name’ dialog appears.

Enter your Dropbox username and click OK. The Authorization Confirmation dialog appears.
Click OK. The Cloud Linking Confirmation window appears.
Enter your Dropbox account email address and password. Click Sign in.
Click Allow. The Cloud Linking Confirmation window displays a success message.
Click Done.
You need to authorize ChemDraw’s interaction with Dropbox folder when you access Dropbox from ChemDraw for the first time. However, you can choose to re-authorize ChemDraw’s interaction with Dropbox folder anytime by checking the Force re-authorization of this application check box in the Enter Dropbox User Name dialog.
For all subsequent sign-in activity to Dropbox:
Navigate to File>Cloud>Dropbox>Sign In. The ‘Enter Dropbox User Name’ dialog appears.
Enter your Dropbox account email address and password.
Click OK. The options to Upload, Download, and Sign out are enabled in ChemDraw.

Uploading Files

You can upload your drawing in the current document window in ChemDraw to your remote Dropbox folder.
To upload a file to Dropbox:
Navigate to File > Cloud>Dropbox>Upload As. The ‘Cloud Upload’ dialog appears.
Specify the name of the file.
Select the file format in which you want to upload the drawing.
Click OK. The ‘Cloud-Select the destination folder’ dialog appears.
Select the folder location where you want to save the file. You can create a new folder by either double-clicking within the folder list column, or double-clicking the Create new folder link.
Specify the new folder name, and click OK.
(Optional) To change the file name, enter the new file name in the File name text field at the bottom of the dialog.
(Optional) You can specify a time limit for the upload process to complete, by specifying a value in ‘seconds’ in the Time out limit text field. You can enter a value between 5 seconds (minimum value) and a maximum value of 120 seconds as the time out limit.
Click Upload.

Downloading Files

You can download files from your remote Dropbox folder to your local machine. When download is complete, the drawing automatically appears in the ChemDraw document window. You can specify the location where the file is saved.
To download a file from your remote Dropbox folder:
Go to File > Cloud>Dropbox>Download. The ‘cloud-select the destination folder’ dialog appears.
Do one of the following:
Double-click the file that you want to download.
Browse to a file that you want to download and click Download.
You can specify a time limit for the download process to complete, by specifying a value in ‘seconds’ in the Time out limit text field. Enter a value between a minimum of 5 seconds and a maximum of 120 seconds as the time out limit.
