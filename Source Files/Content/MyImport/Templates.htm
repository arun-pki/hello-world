Templates

ChemDraw includes a library of templates. You can use a template as a starting point for a new drawing or to add to an existing drawing. A palette is a set of templates.
The templates you define are not limited to atoms and bonds. Templates can also contain captions, colors, boxes, arcs, orbitals, arrows, reaction mechanism symbols, and curves. You can paste pictures from other applications into a template pane. You can download templates from the PerkinElmer Informatics web site and use them. For more information on available templates and user download-able templates, see ChemDraw template files, and BioDraw templates.
To open a template, do one of the following:
Select a template toolbar from the Main toolbar.
Navigate to View>Templates and select a template toolbar from the list.
To add to a drawing using a template:
Select the required template from the template toolbar.
In the document window, click and drag the mouse until the template is oriented the way you need.
To modify the template size, press the <ALT> key while drawing it.
You can also use the templates to add structural features to existing drawings.
To add structural features to existing drawings:
Select the required template from the template toolbar.
Click an atom or bond in the existing drawing to add the template structure.

Customizing Templates

You can modify a template palette to create your own palette. When you create a palette, a set of empty template panes appear above the document window. After you save your palette, it contains the templates you have drawn. You can then select your palette from the palette list in the Templates tool. You can also store templates as groups, or even integral groups, so that they are treated as an individual entity within the documents into which they are dropped.
To create a template palette:
Navigate to File>Open Templates>New Templates.
Click a template pane.
In the document window, draw a new structure.
Click another template pane and draw another structure as desired.
After you draw all the templates you want, go to File>Save.
Save your palette in the ChemDraw Items ChemDraw Templates folder to ensure it appears in the Main toolbar.
In any drawing, select the Template tool on the Main toolbar to use you palette.

Modifying a Template

Navigate to File>Open Templates and choose a palette.
In the palette, click a template pane. The template appears in the document window.
In the document window, edit the template.
Save the template.

To add or delete a palette row or column in a palette:

In the palette, click a cell in the row or column.
Select the appropriate option in the Edit menu.

Adding a Template

To add a template, click in an empty template pane and draw the template in the document window. Navigate to File>Save.

Annotating a Template

Navigate to File>Open Templates and choose a palette.
Right-click on an empty space in a template pane, and select Annotate. The Annotate dialog appears.

Specify a new name for the template in the Content text box.
Click OK.
Click Save. The annotation appears as a tooltip label, when you point to the annotated template.

Deleting a Template

Select the template pane for the template to delete.
In the document window, select the entire drawing.
Navigate to Edit>Clear.
Navigate to File>Save.

Orienting Templates

To make templates you create simple to use, orient the template structure so that you can modify the bonds you use most often.

Resizing Template Panes

To resize the Template panes in the Template panel, drag the lower right corner of the Template panel. If necessary, first click and drag the resize handle.
