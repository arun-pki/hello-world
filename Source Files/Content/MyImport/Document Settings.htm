Document Settings

ChemDraw includes a library of document settings based on those found in many well-known scientific publications. For example, some journals may require submitted articles be formatted with specific margins, fonts, page size, and so on. If you plan to submit your document to a specific journal, open in ChemDraw the style sheet for that journal. You can also create style sheets stationery pads of your own.
The changes you make to the document settings is reflected only in the current document and is not available throughout the application.
To create your own style sheet:
Create a new document.
Enter the settings in the Page Setup, Drawing, Text Settings, and Color palette dialog boxes.
Navigate to File>Save As.
In the Save As dialog box:
Select the ChemDraw style sheetsstationery file format.
Type a name for the document.
Save the template in the ChemDraw Items folder.
Click OKSave.
You can also define/edit document properties in a style sheet. For more information on defining/editing document properties, see Document Properties.
