ChemFinder/Office

You can use CombiChem with ChemFinder for Office to generate libraries of combinatorial experiments. These libraries can be drawn from any type of file that can read, and stored as .mst format databases.
You can enter a CombiChem generic reaction in ChemFinder for Office in either of two ways:
create a generic reaction in either ChemDraw or the ChemDraw ActiveX control.
create a database of generic reactions.

Creating a generic reaction

To create a generic reaction using ChemDraw or the ChemDraw ActiveX control:
Open ChemFinder for Office.
Go to Edit>Edit Structure. Depending on your Preferences setting, this will open ChemDraw or activate the ActiveX control.
Draw a generic reaction. For more information on generic structures, see the ChemDraw User Guide.

Creating a generic reaction database

You can create a database of generic reactions in ChemFinder for Office. ChemFinder for Office can open .cfw files, enabling you to browse through them.
Once you have a generic reaction in the Structure window, go to Search>Enumerate.
The Combi Enumerator window appears, showing the reaction you entered in the Structure window.
(Optional) To add another step to the reaction, click Add Step, edit the reaction in by double clicking in the window, or continue with the reaction as entered.
Click the Analysis tab. An analysis of the reaction steps appears below the reaction display and a new tab is added for each generic reactant in the reaction.
Click the tab for the a generic reactant.
Click the Browse button to select a or SDFile database, or other source of chemical structures such as a collection of files.
Click Search.The search returns all structures in the database file that match the reactant.

Editing the hitlist

You can remove unwanted records from the hit list before continuing:
Click Edit Hitlist. The hitlist opens in ChemFinder.

In ChemFinder, select any unwanted record and go to Record>Omit from List. Repeat this step for each additional unwanted record.
In ChemFinder for Office, click the Enumeration tab. Click Browse and select a database in which to store the results.
Click Go.
Click Save. The results are stored in the database.
To create a database file type a file name.
When you are finished, click OK to exit the Combi Enumerator and return to ChemFinder/Office.
To view the results:
Open ChemFinder.
Go to File> Import>Structures.
Select the database from the Data Import dialog box.

ChemDraw for Excel

You can perform several functions on enumerated experiments. For example, you can calculate properties of the products in an experiment using the property calculation methods. For more information, see the ChemDraw User Guide or ChemDraw for Excel online help.
