Accessing the Document Monomer Importer

You can also import a HELM string from a cdxml file containing it. You can then save it to a library having Read-Write access.
Monomers will only be added to the Document set if they do not appear in any of the existing monomer libraries.
To do so:
Open the CDXML / CDX file. The following message appears if there are monomers present that are not in any of the existing libraries.

Click OK to close the message box or click Open Document Monomer Importer... to display the Document Monomer Importer dialog.

Select a monomer and a destination Monomer Library from the pull-down list and click on Move button.
Clicking on the Move button, moves the monomer from Document set to the destination library without leaving a copy of it in the Document set. You can also edit the definition of a monomer in the Document set and move it to another writable library.
